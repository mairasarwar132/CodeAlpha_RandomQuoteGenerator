# 📖 Overview

QuoteFlow is a **production-ready Flutter application** that displays inspirational quotes through a clean and modern Material 3 interface.

The application is designed with professional Flutter development practices including:

- ✨ Clean Architecture
- 🚀 Riverpod State Management
- 🧭 GoRouter Navigation
- 📱 Responsive UI
- 🎨 Material 3 Design
- 🌙 Light & Dark Theme
- ⚡ Offline-first Experience

Since all quotes are stored locally, **no API, Firebase, or Internet connection is required.**

# 🚀 Getting Started

### 📥 Clone Repository

```bash
git clone https://github.com/mairasarwar132/CodeAlpha_RandomQuoteGenerator.git
```

### 📂 Open Project

```bash
cd CodeAlpha_RandomQuoteGenerator
```

### 📦 Install Dependencies

```bash
flutter pub get
```

### ▶ Run Application

```bash
flutter run
```

---

# 📦 Build APK

```bash
flutter build apk --debug
```
